Nuestra Historia

Hola somos ALT-F4, este grupo esta integrado por Mauro Ulloa, Juan Ignacio Vera, Horacio Quiles y Lucas Sosa

Este proyecto de Integrador Final, presentado para la facultad UTN - SAN RAFAEL.CHEF EN CLICK es un E-Commerce de comida rapida, ideal para vender un producto de comida rapida.

Los lenguajes utilizados en este proyecto fue, por la parte de base de datos utilizamos SQL, para el backend utilizamos Node y para el Front utilizamos Css, JavaScript
